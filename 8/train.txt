출발역: 		서울, 대전, 동대구, 부산
도착역: 		서울, 대전, 동대구, 부산
기차종류: 	KTX, 새마을
좌석유형: 	일반, 특실

# KTX인 경우에만 특실이 존재한다. -> 새마을은 특실이 없다.
IF [기차종류] = "새마을" THEN [좌석유형] <> "특실";
# 출발역과 도착역은 다르다. 
IF [출발역] = "서울" THEN [도착역] <> "서울";
IF [출발역] = "대전" THEN [도착역] <> "대전";
IF [출발역] = "동대구" THEN [도착역] <> "동대구";
IF [출발역] = "부산" THEN [도착역] <> "부산";
